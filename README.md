# COVID-19 Report Analysis using Power BI

## 📌 Project Overview

COVID-19 Report Analysis is an interactive Business Intelligence (BI) project developed using **Microsoft Power BI** to analyze global COVID-19 pandemic data. The project transforms raw data from the **Our World in Data (OWID)** dataset into insightful and easy-to-understand visual dashboards, enabling users to explore the spread of the pandemic, vaccination progress, testing statistics, and mortality trends.

The dashboard is designed to help researchers, students, healthcare professionals, and policymakers understand the impact of COVID-19 through dynamic visualizations and data-driven insights.

## 🚀 Features

* Interactive Power BI dashboard
* Global and regional COVID-19 analysis
* Total confirmed cases, deaths, recoveries, and vaccinations
* Death rate and vaccination rate analysis
* Time-series trend visualization
* Country/continent-wise comparison
* KPI Cards for key pandemic statistics
* Interactive filters (Year, Month, Date, Continent, Country)
* Maps, Line Charts, Bar Charts, Pie Charts, and Scatter Plots
* Dynamic DAX calculations for custom metrics

## 🛠️ Technologies Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel/CSV
* Our World in Data (OWID) COVID-19 Dataset

## 📊 Workflow

1. Collect COVID-19 dataset from OWID.
2. Import data into Power BI.
3. Clean and transform data using Power Query.
4. Create relationships and data model.
5. Build DAX measures for KPIs.
6. Design interactive dashboards.
7. Analyze pandemic trends and generate insights.

## 📈 Dashboard Insights

* Daily and cumulative COVID-19 cases
* Total deaths and mortality trends
* Vaccination progress across countries
* Testing statistics
* Continent-wise comparison
* Pandemic growth over time
* Correlation between vaccination and death rates

## 🎯 Objectives

* Visualize COVID-19 data effectively.
* Convert raw healthcare data into meaningful insights.
* Support data-driven decision-making.
* Demonstrate the capabilities of Microsoft Power BI in real-world healthcare analytics.

## 📂 Dataset

The project uses the publicly available **Our World in Data (OWID) COVID-19 Dataset**, which contains worldwide COVID-19 statistics collected from trusted official sources.
